# Zyra Sneakers

A modern, interactive sneaker showcase website built with Next.js, React Three Fiber, and Tailwind CSS. Features stunning 3D visualizations, smooth animations, and a responsive design to showcase premium footwear.

## Features

- 🎨 **Modern Design** - Clean, contemporary UI with gradient backgrounds and vibrant accent colors
- 🎬 **Smooth Animations** - Framer Motion-powered transitions and interactions
- 🌐 **3D Graphics** - React Three Fiber integration for immersive 3D shoe visualization
- 📱 **Responsive** - Fully responsive layout that works across all devices
- ⚡ **Performance Optimized** - Built with Next.js 16 for fast load times
- 🎯 **TypeScript** - Type-safe codebase for better developer experience

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) 16.2.6
- **UI Library**: React 19.2.4
- **Styling**: Tailwind CSS 4 + PostCSS
- **3D Graphics**: Three.js + React Three Fiber 9.6.1 + Drei 10.7.7
- **Animations**: Framer Motion 12.40.0
- **Language**: TypeScript 5
- **Linting**: ESLint 9

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn package manager

### Installation

1. Clone the repository or navigate to the project directory:
```bash
cd zyra-sneakers
```

2. Install dependencies:
```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Build & Production

Build the application for production:

```bash
npm run build
```

Start the production server:

```bash
npm start
```

### Linting

Check code quality:

```bash
npm run lint
```

## Components

- **Navbar** - Main navigation component
- **HeroSection** - Eye-catching hero banner
- **ShoeSection** - Individual shoe showcase with details and features
- **MarqueeStrip** - Animated scrolling marquee strip
- **FeaturesBadge** - Badge component for displaying features
- **Footer** - Footer component

## Development Notes

- The project uses Next.js App Router for routing and layout
- 3D shoe visualizations are powered by Three.js and React Three Fiber
- Animations are handled by Framer Motion for smooth, performant transitions
- Tailwind CSS is used for all styling with custom color schemes per shoe
